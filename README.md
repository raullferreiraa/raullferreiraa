## 🚀 Projeto em Destaque

### [Garagem Digital](https://github.com/raullferreiraa/garagem-digital) 🚘

Aplicação web Full Stack criada para catalogar, documentar e exibir projetos automotivos.

O projeto começou como um CRUD de veículos e evoluiu para a base inicial de uma plataforma automotiva com características sociais, incluindo autenticação de usuários, garagem pessoal, feed geral de projetos e controle de propriedade.

**Principais recursos:**

- Cadastro e login de usuários;
- Senhas armazenadas com hash;
- Registro de projetos automotivos com ficha técnica;
- Upload de imagens com validação;
- Campo de história/modificações do projeto;
- Feed geral com todos os projetos cadastrados;
- Aba “Meus projetos” com a garagem pessoal do usuário logado;
- Edição e exclusão permitidas apenas ao dono do projeto;
- Visualização detalhada em modal;
- Filtros por modelo, suspensão e aro.
